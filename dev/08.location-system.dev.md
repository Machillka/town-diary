## 代码解释

LocationSystem 是只读拓扑，不保存 Agent 位置。它提供：

- 地点存在性查询。
- 双向连接查询。
- 当前 time block 开放状态。
- 公共、私人和核心叙事地点分类。
- 不可变地点状态快照。

开放状态由 WorldClock 的统一 TimeBlock 推导，地点不会自行推进时间。

## 关键分析与取舍

- 区分 `is_public` 与 `is_core_narrative`，避免把小说家家错误标记为公共地点。
- 拒绝单向连接，MVP 不引入单向道路语义。
- 私人住处使用 `always` 开放，公共地点使用明确时间段。
- LocationSystem 不负责判断 Agent 是否允许进入私人地点；该权限属于后续 ActionValidator。
