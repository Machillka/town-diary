## 分析与决策

- 世界运行是项目本体，小说家与 Writing 都是下游消费者。
- 小说家不是导演或全知视角，只是具有主观性的 Agent camera。
- 客观事实、主观认知和文本表达必须使用不同数据层。
- 原始材料中缺失的天气步骤被补入核心路线。
- Habit、最小传闻和 ExperienceLog 被确认为 MVP-0.1 必需能力。
- Agent 当前地点双真源、单阶段 tick 和 Writing 读取 WorldLog 被识别为最高风险。
